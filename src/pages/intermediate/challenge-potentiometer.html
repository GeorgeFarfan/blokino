Potenciometro - Preparación

Antes de iniciar el desafío se debe crear un Potenciometro y un Led. El nombre del Potenciometro debe ser POTENCIOMETRO_LED . Para crear un potenciometro se debe usar el bloque Crear Potenciometro, el valor del pin de datos debe ser A3. El nombre del LED debe ser LED y su pin de datos: 13.

Instrucciones

Prueba 1

  • Consigna: Animate a ajustar la velocidad de parpardeo del LED. Se debe crear un procedimiento llamado AjustarLED, donde se ajuste la velocidad del parpadeo del LED.

Pista: Sabías para ajustar la velocidad de parpadeo de un LED se puede usar el bloque Girar.

Aviso: Si tienes muchos dispositivos conectados. No te olvides de configurarlos, para poder ejecutar los programas de Blokino.

¿Cómo armo mi circuito?

Para poder completar el desafío de los Potenciometros, se debe tener un circuito bien armado. Hay varias manera de armar estos circuitos válidos, como se ven en las siguientes imágenes:

Circuito - Armado

En la siguiente imagen se puede ver como se debe armar el circuito usando una protoboard y placa Ardunio UNO. Como se ve en la siguiente imagen:

En la imagen se definen las entradas que necesita el POTENCIOMETRO, una de las entradas se usa para la energia GND, otra para la entrada de los 5V, una resistencia con esos colores especificos y una entrada ANALOGICA que en este caso es el número A3. Este circuito se debe repetir si se quieren agregar más potenciometros, se recomienda usar los mismos colores de cables para no generar confusión.
