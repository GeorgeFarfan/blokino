BOTONES, PULSADORES, INTERRUPTORES - Preparación

Antes de iniciar el desafío, se deben crear un led, botones, pulsador y un interruptor. Para crearlos, se debe usar el bloque funcional Crear Variable. El led debe llamarse LED y su número de pin debe ser 13. El primer boton debe llamarse boton_encender_led con el número de pin 2 y el segundo boton debe llamarse boton_detener_led con el número de pin 8. El pulsador debe llamarse pulsador_detener_led con el número de pin 7. Y por último el interruptor debe tener el siguiente nombre interruptor_led con el número de pin 12.

Instrucciones

Prueba 1

  • Consigna: Animate a hacer parpadear un LED con botones y funciones. Crear un procedimiento llamado EncenderLED que cuando se presione el boton_encender_led se encienda con un parpadeo de 3 milisegundos. Crear un procedimiento llamado DetenerLED que cuando se presione el boton_detener_led se detenga el parpadeo del LED.

Pista: Se pueden crear nuevas acciones en Procedimientos definiendo nuevos bloques que incluyan otras acciones. Para crear procedimientos, puedes usar el bloque Crear Procedimiento

Primera manera de armar el circuito
